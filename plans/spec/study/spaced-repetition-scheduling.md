# Spec: spaced-repetition-scheduling

## 한 줄 정의
사용자가 학습 카드를 복습할 때 평가(Again, Hard, Good, Easy)에 따라 다음 복습 일정이 자동으로 계산된다.

## 진입점
- 복습 세션에서 카드를 평가할 때 자동 발동
- 복습 대기열 버튼 클릭으로 복습 세션 시작

## 사용자 흐름
1. 사용자가 복습 대기열을 연다.
2. 오늘 복습할 카드가 순서대로 표시된다.
3. 사용자가 카드를 확인하고 자신의 기억 수준을 평가한다.
   - Again: 기억나지 않음 → 곧 다시 복습
   - Hard: 어렵게 기억 → 짧은 간격 후 복습
   - Good: 잘 기억 → 표준 간격 후 복습
   - Easy: 매우 쉽게 기억 → 긴 간격 후 복습
4. 다음 복습 날짜가 자동으로 계산되어 저장된다.
5. 복습 대기열에 다음 카드가 표시된다.

## 성공 조건 (Acceptance Criteria)
- [ ] SM-2 알고리즘에 따라 난이도 계수(E-Factor)가 업데이트된다.
- [ ] Again 평가 시 같은 세션 내에 곧 다시 표시된다.
- [ ] Hard 평가 시 이전 간격보다 짧은 간격으로 예약된다.
- [ ] Good 평가 시 표준 간격(이전 간격 * E-Factor)으로 예약된다.
- [ ] Easy 평가 시 표준 간격보다 긴 간격으로 예약된다.
- [ ] 연속 정답 시 복습 간격이 점진적으로 증가한다.
- [ ] 연속 오답 시 복습 간격이 단축된다.
- [ ] 오늘 예정된 카드가 대기열 상단에 표시된다.
- [ ] 과거 예정이었지만 미루어진 카드도 대기열에 포함된다.
- [ ] 복습 통계(정답률, 평균 간격, 누적 복습 횟수)가 업데이트된다.

## 실패 / 취소 흐름
- 복습 대기열이 비어 있으면 "모든 카드를 복습했습니다" 메시지가 표시된다.
- 세션 중 Escape를 누르면 진행 상태가 저장되고 나중에 이어서 할 수 있다.

## 경계 / 예외
- 같은 카드를 같은 세션에서 여러 번 Again해도 무한 루프에 빠지지 않는다.
- 대량의 카드(1,000장 이상)가 있어도 대기열 계산이 즉시 수행된다.
- 보류(bury)된 카드는 다음 날까지 대기열에서 제외된다.
- 일시정지(suspended)된 카드는 대기열에서 제외된다.

## 범위 외
- 카드 생성/편집은 별도 spec으로 분리.
- FSRS 알고리즘 힌트 적용은 별도 spec으로 분리.
