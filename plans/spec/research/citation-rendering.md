# Spec: citation-rendering

## 한 줄 정의
사용자가 참고문헌을 추가하면 선택한 인용 스타일(APA, MLA, Chicago, BibTeX, RIS)에 따라 인용과 참고문헌 목록이 자동으로 서식화된다.

## 진입점
- 인스펙터 패널 > 인용 탭: 인용 스타일 선택
- Ctrl+Shift+C: 인용 포맷 순환 전환
- 인용 결과 > 삽입 버튼: 문서에 인용 삽입
- 헤더 > 내보내기: 참고문헌 목록 내보내기

## 사용자 흐름
1. 사용자가 논문을 선택하고 인스펙터 패널의 인용 탭을 연다.
2. 인용 스타일(APA, MLA, Chicago, BibTeX, RIS)을 선택한다.
3. 선택한 스타일에 맞게 인용 미리보기가 즉시 업데이트된다.
4. "삽입" 버튼을 누르면 서식화된 인용이 클립보드에 복사된다.
5. 참고문헌 목록을 Plain Text, Markdown, HTML 포맷으로 내보낼 수 있다.

## 성공 조건 (Acceptance Criteria)
- [ ] APA 7판 스타일로 인용이 올바르게 서식화된다.
- [ ] MLA 9판 스타일로 인용이 올바르게 서식화된다.
- [ ] Chicago 17판(저자-연도) 스타일로 인용이 올바르게 서식화된다.
- [ ] BibTeX 포맷으로 인용이 출력된다.
- [ ] RIS 포맷으로 인용이 출력된다.
- [ ] 스타일 전환 시 미리보기가 즉시 업데이트된다.
- [ ] 참고문헌 목록이 알파벳 순으로 정렬된다.
- [ ] 저자명, 연도, 제목, 학회/저널, DOI가 정확히 표시된다.
- [ ] 공동 저자(2인 이상)가 올바르게 표시된다.
- [ ] 클립보드 복사가 서식을 포함하여 동작한다.

## 실패 / 취소 흐름
- 메타데이터가 불완전한 논문은 가능한 필드만으로 인용을 생성한다.
- CSL 스타일 검증 실패 시 에러 메시지가 표시된다.

## 경계 / 예외
- 같은 논문이 여러 컬렉션에 있어도 인용은 한 번만 생성된다.
- 비영어 논문(한글, 일본어 등)의 인용도 지원된다.
- 온라인 출판물(URL, 접근일)의 인용이 올바르게 서식화된다.

## 범위 외
- 워드 프로세서 플러그인 연동은 별도 spec으로 분리.
- CSL 스타일 편집기는 별도 spec으로 분리.
